About Superhero-pedia

Superhero-pedia is a learning project that explores the Superhero API. Search by name or ID, flip through powerstats, and discover fun facts for each hero or villain.

Built with vanilla JavaScript, Bootstrap 5, and an Express backend proxy to keep requests safe and fast. Data provided by the open-source Superhero API.

Built by Alberto Cisneros and Juan Francisco Luna | Superhero data courtesy of Superhero API
